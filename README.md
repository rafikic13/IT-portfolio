# IT Portfolio

## Overview

This repository contains a collection of technical projects and labs developed to demonstrate practical skills in information technology, system administration, and infrastructure management.

The projects included here focus on real-world tasks commonly performed by IT technicians, help desk specialists, and system administrators. The goal of this portfolio is to showcase hands-on experience with scripting, system tools, and administrative automation.
