# Virtualization Projects

This section contains virtualization labs completed during my IT program at Collège de Rosemont.

## ESXi Cluster with vCenter

Objective:
Create a VMware ESXi cluster with two hosts managed by vCenter.

Technologies used:
- VMware ESXi
- vCenter
- DRS
- vSphere HA
- vSAN

Tasks completed:
- Installed two ESXi hosts
- Created a cluster in vCenter
- Enabled DRS, HA and vSAN
- Migrated a Windows 10 virtual machine between hosts

Result:
The virtual machine was successfully migrated between ESXi hosts using vMotion.

## Hyper-V Virtual Network

Objective:
Configure a virtual network using Hyper-V.

Technologies used:
- Windows Server 2022
- Hyper-V
- NAT networking

Tasks completed:
- Installed Hyper-V
- Created two virtual machines (DC22 and W10)
- Created internal and external virtual switches
- Configured NAT for internet access

Result:
The Windows 10 machine successfully accessed the internet through the NAT network.
